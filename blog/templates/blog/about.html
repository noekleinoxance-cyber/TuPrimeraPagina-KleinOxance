Sobre el proyecto

Este proyecto fue desarrollado por Noé Klein Oxance como parte del curso de Python y Django.

La aplicación simula una Mesa de Ayuda IT donde se pueden registrar casos de soporte técnico, documentar soluciones y consultar procedimientos.

El sistema permite:

  • Crear casos de soporte
  • Buscar soluciones registradas
  • Agregar comentarios
  • Administrar contenido desde el panel de Django
